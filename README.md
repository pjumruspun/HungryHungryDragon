# HungryHungryDragon
Spaceship (dragon) shooter game done purely in Assembly (68k) code

## How to install
1. Download the Easy68k simulator from http://www.easy68k.com/
2. Clone or download this repository
3. Open main.S68 with the Easy68k you just downloaded, you'll see this pops up, press F9 to run
